# Generate subject-specific scores given estimates of mean and eigenfunctions

Internal function to calculate scores

## Usage

``` r
get_score(yi, Mu, Phi, sigma, n.comp)
```

## Arguments

- yi:

  one function input data

- Mu:

  estimated mean

- Phi:

  matrix of estimated fpcs

- sigma:

  residual variance

- n.comp:

  number of estimated fpcs
