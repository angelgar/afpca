# Function to orthonormalize (or orthogonalize) scores

Internal function to orthonormalize or orthogonalize scores

## Usage

``` r
orthonormal_step(c_mat, orthogonalize_scores, normalize.scores)
```

## Arguments

- c_mat:

  matrix of scores

- orthogonalize_scores:

  logical; wether to orthogonalize scores

- normalize.scores:

  logical; wether to normalize scores
