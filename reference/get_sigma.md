# Estimate residual variance

internal function to estimate resisual variance

## Usage

``` r
get_sigma(data, coef, Theta, c_mat, N.Unp.Basis, nbs)
```

## Arguments

- data:

  input data

- coef:

  estimated spline coefficients

- Theta:

  spline basis matrix

- c_mat:

  scores matrix

- N.Unp.Basis:

  number of unpenalized spline basis

- nbs:

  total number of spline basis
